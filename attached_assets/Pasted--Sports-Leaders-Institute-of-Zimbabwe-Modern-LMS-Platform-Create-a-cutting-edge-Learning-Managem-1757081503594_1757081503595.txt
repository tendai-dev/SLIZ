# Sports Leaders Institute of Zimbabwe - Modern LMS Platform

Create a cutting-edge Learning Management System for the Sports Leaders Institute of Zimbabwe with the following specifications:

## Core Technology Stack
- **Frontend**: Next.js 14 with TypeScript and Tailwind CSS
- **Authentication**: Clerk for complete user management
- **Database**: PostgreSQL with Prisma ORM
- **SCORM Support**: Built-in SCORM 1.2 and 2004 compliance
- **Deployment**: Vercel-ready configuration

## Authentication & User Management
- Implement Clerk authentication with custom sign-up/sign-in forms
- User roles: Admin, Instructor, Student
- Profile management with avatar upload
- Social login options (Google, LinkedIn)
- Custom onboarding flow for sports leadership context

## UI/UX Design Requirements
Create a **futuristic sports-themed design** with:
- Dark mode with neon accent colors (electric blue, lime green)
- Glassmorphism cards and components
- Smooth micro-interactions and animations
- 3D hover effects on course cards
- Modern typography (Inter/Poppins)
- Responsive design for all devices
- Loading states with skeleton UI
- Toast notifications for user feedback

```

```
